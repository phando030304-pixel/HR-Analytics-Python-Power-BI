## Overview

This project was completed as part of the Google Advanced Data Analytics Professional Certificate Capstone.

The objective of this project is to analyze employee attrition patterns and identify the factors that contribute to workforce turnover. Through exploratory data analysis, statistical investigation, and business intelligence reporting, the project aims to generate actionable insights that support employee retention and workforce management strategies.

Using Python for data preparation and analysis and Power BI for dashboard development, this project follows an end-to-end analytics workflow including data cleaning, feature exploration, visualization, insight generation, and business recommendation development.

The analysis focuses on:

- Employee attrition
- Job satisfaction
- Salary levels
- Work accidents
- Promotion history
- Department performance

The final result includes analytical insights, interactive dashboards, and strategic recommendations designed to support HR decision-making and employee retention initiatives.

## Key Questions

1. What factors contribute most to employee attrition?
2. How does job satisfaction influence turnover?
3. Does salary level affect employee retention?
4. Which departments experience the highest attrition rates?
5. What actions can organizations take to reduce turnover?

---

## Key Insights

- Employees with lower satisfaction levels are more likely to leave.
- Low-salary employees show higher attrition rates.
- Employees with excessive workloads are more likely to resign.
- Promotion opportunities appear to improve retention.
- Attrition rates vary significantly across departments.

---

## Recommendations

### 1. Improve Employee Satisfaction

Develop programs focused on employee engagement and workplace well-being.

### 2. Review Compensation Strategies

Evaluate salary structures for high-turnover employee groups.

### 3. Monitor Workload Distribution

Reduce burnout risks by balancing project assignments and working hours.

### 4. Strengthen Career Development

Create promotion and professional development opportunities to improve retention.
